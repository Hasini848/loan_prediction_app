# Loan Prediction App 🚀

## 📌 Project Overview

This project is a **Machine Learning-based Loan Prediction System** built using **XGBoost Regressor**.
It predicts loan-related outcomes based on user inputs like age, income, city, and employment type.

---

## 🧠 Model Details

* Algorithm: **XGBoost Regressor**
* Training Accuracy (R²): ~90%
* Testing Accuracy (R²): ~40%
* Issue handled: **Overfitting reduced using tuning**

---

## 📊 Features Used

* Age
* Income
* Loan Amount
* Credit Score
* Number of Transactions
* Annual Spend
* Date (Day, Month, Year)
* City (Encoded)
* Employment Type (Encoded)
* Loan Type (Encoded)

---

## 🚀 How to Run Locally

1. Install dependencies:

   ```
   pip install -r requirements.txt
   ```

2. Run the app:

   ```
   streamlit run app.py
   ```
